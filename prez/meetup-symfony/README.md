Je me rends compte que leurs utilisations est plus ou moins connu et puis celà permet de détailler quelques concepts liés à ces outils. ## Informations générales
 
 * Outils en PHP (généralement) mais aussi en Javascript
 * Facile à déployer (Merci Composer et Phar)
    
    ```
    $ cd /usr/local/bin 
    $ sudo wget outil.phar
    $ sudo chmod a+x outil.phar
    $ sudo ln -s outil.phar outil

    ```
  
 * A utiliser sans modération
 
Note: Je ne rentrerais pas dans les détails des outils. Je vais suivre une grille pour parler de l'outil, de son utilisation et des actions qu'ils peuvent vous aider à faire. De manière générale ces outils sont en PHP, il sont téléchargeable sous forme de fichier phar qu'il suffit de placer sur votre répertoire /usr/local/bin et vous pouvez commencer à les utiliser. Selon les cas certains outils utilisent composer ou npm et ce n'est guère plus compliquer.
